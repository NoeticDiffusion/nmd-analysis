# How to write the EAP result

Use this wording:

During cognitive control, higher respiratory anchoring was associated with lower global manifold mobility after partialling HRV, reaching nominal significance and marginal FDR correction. These findings are consistent with, but do not establish, the hypothesis that interoceptive anchoring moderates local neural flow geometry.

This keeps it honest.
